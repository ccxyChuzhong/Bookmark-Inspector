# 🎬 Bookmark Inspector 功能演示（v3.0）

## 🌟 v3.0 新功能速览

### 🏷️ 书签状态徽标

书签列表会在每一条 URL 右侧显示状态：

| 徽标 | 颜色 | 含义 |
| --- | --- | --- |
| 可用 | 绿色 | 上次检测可访问 |
| 失效 | 红色 | 上次检测不可访问 |
| 检测中 | 蓝色 | 当前正在被检测 |
| 未检测 | 中性灰 | 从未检测过 |

**关键改进**：v2.x 中未检测过的书签会被错误地标记为"检测中"，v3.0 已修复并新增"未检测"徽标。文件夹项不显示徽标。

### 🗑️ 行内删除

每条书签的行尾追加一个图标按钮：

```
┌──────────────────────────────────────────────┐
│ 🌐  Vue.js 官网                              │
│     https://vuejs.org/    可用  3 分钟前  🗑 │ ← 悬停时浮现
└──────────────────────────────────────────────┘
```

- 默认 `opacity: 0`，不占视觉重量
- 鼠标悬停在该行时浮现到 `opacity: 0.7`
- 按钮本身 hover 时变为危险红色
- 点击后弹出二次确认，确认即从浏览器和列表中删除

### 📊 进度条修复

**v2.x 已知问题**：检测过程中进度条不显示。
**根因**：`updateSectionVisibility()` 在检测中误将整个结果区设为 `display: none`，进度条作为子元素一起被隐藏。
**修复**：检测中或已有结果时都让结果区可见。

### ⚙️ 紧凑配置页

`settings.html` 的留白被统一压缩，常规屏幕上 **代理 / 并发 / WebDAV** 三大模块一屏可见。

| 项目 | 之前 | 之后 |
| --- | --- | --- |
| Wrapper 上下边距 | 48px | 20px |
| Section 内边距 | 22px / 24px | 12px / 18px |
| 输入框高度 | 38px | 32px |
| WebDAV 按钮排布 | flex-wrap 多行 | 4 列等宽 grid |

## 📥 智能导入功能（继承自 v2.0）

### 🎯 功能概述
新版本的书签导入功能提供了更加灵活和智能的导入选项，让用户可以根据需要选择不同的导入方式。

### 🔧 功能特点

#### 1. 模态对话框设置
- 点击"选择文件导入"按钮后，会弹出设置对话框
- 用户可以在导入文件之前先配置导入选项
- 提供清晰的操作流程和用户引导

#### 2. 文件夹选择
- **目标文件夹选择**：可以选择导入到任意书签文件夹
- **动态加载**：自动加载用户的所有书签文件夹
- **默认选项**：默认选择书签栏作为导入目标

#### 3. 导入模式选择
- **平铺模式**：将文件中的所有书签提取出来，导入到选中文件夹的根目录
- **保持结构**：完整保持文件中的原有文件夹层次结构
- **动态说明**：根据选择的模式显示相应的说明文字

### 📋 操作流程演示

#### 步骤1：点击导入按钮
```
用户界面 → 点击"导入书签"按钮
```

#### 步骤2：配置导入设置
```
模态对话框弹出 → 选择目标文件夹 → 选择导入模式
```

#### 步骤3：确认并选择文件
```
点击"选择文件"按钮 → 选择文件 → 开始导入
```

### 💡 使用建议

#### 何时使用平铺模式
- 整合多个来源的书签到一个文件夹
- 不需要保持原有的文件夹结构
- 希望所有书签都在同一层级

#### 何时使用保持结构模式
- 完整迁移书签数据
- 保持原有的组织结构
- 需要维护书签的分类体系

### v3.0 端到端测试清单

1. **状态徽标**
   - 全新加载扩展 → 选择一个文件夹 → 确认每条书签显示「未检测」
   - 点击「检测网址可用性」 → 检测过程中确认显示「检测中」
   - 检测完成后确认显示「可用」或「失效」

2. **进度条**
   - 检测过程中进度条应实时显示，百分比和 `x/y` 计数同步更新

3. **行内删除**
   - 悬停一条书签 → 行尾浮现垃圾桶图标
   - 点击 → 确认对话框 → 确认后该项即时从列表移除，列表计数同步减少

4. **紧凑设置页**
   - 打开 `settings.html`，在 1280×720 及以上的常规屏幕上代理 / 并发 / WebDAV 应一屏可见

5. **WebDAV**
   - 填写有效配置 → 测试连接 → 备份 → 在另一台机器或清空数据后恢复
